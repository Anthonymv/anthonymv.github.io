Caso: Arquitectura BI en Azure

Esta solución demuestra cómo automatizar reportes médicos utilizando servicios en la nube de Azure. Basado en datos públicos, este diseño muestra una arquitectura moderna y segura para visualización en tiempo real.

🔙 Volver a Arquitectura Cloud

📘 Descripción del Caso

Este caso se enfoca en la automatización de reportes médicos utilizando componentes nativos de Azure, permitiendo escalar y monitorear KPIs en tiempo real con una arquitectura segura y desacoplada.

📌 Objetivos

  • Automatizar el flujo de datos desde origen hasta dashboard.
  • Usar orquestadores y almacenes escalables.
  • Asegurar la arquitectura con buenas prácticas.

🔧 Servicios utilizados

  • Azure SQL: Base de datos operativa.
  • API REST: Servicios externos con datos complementarios.
  • Azure Data Factory: Ingesta y transformación de datos.
  • Azure Data Lake Gen2: Almacenamiento de staging.
  • Azure Synapse Analytics: Consulta de datos.
  • Power BI (DirectQuery): Visualización de KPIs en tiempo real.
  • Azure Key Vault: Gestión de secretos y credenciales.
