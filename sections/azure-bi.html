Arquitectura BI en Azure

Diseño de una solución escalable para automatización de reportes médicos usando servicios nativos de Azure. Toda la documentación se encuentra integrada visualmente en esta página.

🔙 Volver a Arquitectura Cloud

📌 Objetivo

Automatizar reportes clínicos y operacionales de una plataforma de salud digital utilizando servicios Azure.

🧱 Componentes Arquitectónicos

  • Origen de Datos: Azure SQL Database y endpoints REST de pacientes.
  • Orquestación: Azure Data Factory (pipelines, triggers, linked services).
  • Almacenamiento: Azure Data Lake Gen2 como staging y Synapse Analytics para modelado.
  • Visualización: Power BI conectado por DirectQuery para reportes en tiempo real.
  • Seguridad: Azure Key Vault + RBAC (control por roles).
