🔙 Volver a Arquitectura Cloud

📌 Objetivo del Proyecto

Diseñar e implementar una arquitectura en Azure para analizar indicadores de salud pública utilizando datos abiertos de Chile.

1️⃣ Adquisición de Datos Públicos

  • 🔗 Fuente: Portal de Datos Abiertos de Chile, DEIS MINSAL, FONASA.
  • 📂 Formato: CSV descargado y subido a Azure Blob Storage.

2️⃣ Integración con Azure Data Factory

  • 🛠️ Pipeline ADF: lectura desde Blob Storage y carga a Data Lake Gen2.
  • 🔄 Transformaciones: limpieza y estructuración con Data Flows.

3️⃣ Modelado en Azure Synapse Analytics

  • 📊 Modelo: Diseño estrella para consultas analíticas.
  • ⚙️ Motor: Serverless SQL pool conectado al Data Lake.

4️⃣ Visualización en Power BI

  • 📈 Conexión: DirectQuery a Synapse para reportes en tiempo real.
  • 🧩 Paneles: Mortalidad, atenciones, patologías, regiones.

🔐 Seguridad y Gobernanza

  • 🧾 Azure Key Vault para almacenar secretos.
  • 🔐 RBAC para controlar acceso por roles.
  • 🛰️ Azure Monitor para trazabilidad.
